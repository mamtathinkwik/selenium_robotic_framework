# ------ SETUP WITHOUT VIRTUAL ENVIRONMENT / WITHOUT PIPENV------

### FOLLOW BELOW STEPS -

#### INSTALL LIBRARIES
```pip install -r requirements.txt```

## ------ SETUP WITH VIRTUAL ENVIRONMENT / WITH PIPENV ------

### FOLLOW BELOW STEPS -

#### INSTALL PIPENV
```pip install pipenv```

#### INSTALL LIBRARIES
```pipenv install```
